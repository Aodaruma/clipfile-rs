# Security policy

`clipfile` parses untrusted binary input and treats malformed-file handling as
a security concern. Please report potential denial-of-service, memory-safety,
path-traversal, or unexpected-write issues privately through GitHub Security
Advisories when the repository is published.

Only the latest released version will receive security fixes during the early
development phase. Until a stable release exists, all APIs and file-format
coverage should be considered experimental.
