# 2Dカメラ追加差分の手動作成手順

2Dカメラの基本構造は実装済みです。この手順は、snapshot内に残る未命名wordと各値の単位を追加検証する場合だけ実施します。急ぎではありません。

## 保存先と共通条件

- 匿名fixtureだけを使い、Git管理対象外の `tester/data/manual_generated/` に保存する。
- 既存のカメラfolder追加直後のfixtureを毎回開き直し、変更を累積させない。
- 同じ先頭以外のフレームを選び、記載した項目だけを変更する。
- 作品名、人物名、実案件名、本文、既存素材は追加しない。

## 作成する3ファイル

1. 拡大率だけを100から120へ変更し、`camera_manual_scale.clip` として別名保存する。
2. 元fixtureを開き直し、回転角だけを0から15度へ変更して、`camera_manual_rotation.clip` として別名保存する。
3. 元fixtureを開き直し、不透明度だけを100から80へ変更して、`camera_manual_opacity.clip` として別名保存する。

各ファイルでは、タイムラインに変更項目のキーが表示されたことを確認してください。位置、回転中心、出力サイズ、他の値は変更しません。

## 解析時の扱い

生成物はGitへ追加しません。公開文書には構造差だけを集計し、実体の保存場所、UUID、ユーザーパス、個人・作品情報は転記しません。
